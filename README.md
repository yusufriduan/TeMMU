# TeMMU
## What is TeMMU
TeMMU is a Collaborative platform that provides matchmaking between mentors and mentees.
This gives STEM students the opportunity to collaborate with each other as they can find suitable mentors and peers.
Our web application also allows operations management via shared notes and chats.

## Features
1. Provide a workspace to be a centralised platform for collaborating, planning and chatting
2. Post discussion forums to request for help
3. Matchmaking to recommend students suitable mentors
4. Mentee management for mentors
